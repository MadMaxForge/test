# Правый Берег — VK Automation Bot

Автоматизация контента для VK-сообщества "Правый Берег" (недвижимость, Нижегородская область).

## Возможности

- **Генерация постов**: AI создаёт текст + фото-обложка с заголовком (затемнение + текст поверх)
- **Генерация рилсов**: AI скрипт + TTS озвучка (ElevenLabs) + видео-нарезки + субтитры
- **Telegram-бот**: модерация контента (одобрить / отклонить / перегенерировать)
- **Загрузка медиа**: фото и видео через TG-бот прямо с телефона
- **Парсинг конкурентов**: автоматический сбор тем из групп конкурентов (tierra_moscow, zagorodnyezemli)
- **Расписание**: Вт/Чт 19:00, Сб/Вс 10:00 МСК
- **Антиповтор**: система отслеживает использование медиа и тем

## Структура

```
pravyi-bereg/
├── main.py              # Точка входа (бот + планировщик)
├── config.py            # Конфигурация из .env
├── db.py                # SQLite база данных
├── pipeline.py          # Пайплайн: генерация → очередь → публикация
├── scheduler.py         # APScheduler расписание
├── content/
│   ├── generator.py     # Генерация текста через OpenRouter
│   ├── topics.py        # Банк тем (50+ тем по категориям)
│   ├── parser.py        # Парсинг конкурентов VK
│   └── validator.py     # Валидация контента
├── media/
│   ├── photo_overlay.py # Фото: затемнение + текст заголовка
│   └── video_builder.py # Видео: FFmpeg сборка рилсов
├── tts/
│   └── voice.py         # TTS: ElevenLabs / edge-tts
├── bot/
│   └── handlers.py      # TG-бот: команды и обработчики
├── vk/
│   └── api.py           # VK API: публикация постов/видео
├── .env.example         # Шаблон переменных окружения
└── requirements.txt     # Python зависимости
```

## Установка

```bash
# 1. Создать виртуальное окружение
python3 -m venv venv
source venv/bin/activate

# 2. Установить зависимости
pip install -r requirements.txt

# 3. Настроить переменные окружения
cp .env.example .env
# Заполнить .env своими ключами

# 4. Запустить
python main.py
```

## Расписание публикаций

| День | Время (МСК) | Тип контента |
|------|-------------|-------------|
| Вторник | 19:00 | Пост (фото + статья) |
| Четверг | 19:00 | Рилс (видео + озвучка) |
| Суббота | 10:00 | Пост (фото + статья) |
| Воскресенье | 10:00 | Рилс (видео + озвучка) |

Контент генерируется за 2 часа до публикации и отправляется на проверку в Telegram.

## Команды TG-бота

- `/start` — Приветствие и список команд
- `/status` — Статус системы (медиа, очередь, публикации)
- `/queue` — Очередь контента
- `/generate` — Сгенерировать пост вручную
- `/topics` — Статистика по темам
- `/stats` — Общая статистика
- Отправка фото/видео — автоматическое сохранение как исходников

## Технологии

- **OpenRouter** (Gemini 2.0 Flash) — генерация текста
- **ElevenLabs** — озвучка для рилсов
- **Pillow** — обработка изображений
- **FFmpeg** — сборка видео
- **python-telegram-bot** — Telegram бот
- **APScheduler** — планировщик задач
- **SQLite** — база данных (очередь, история, темы)
