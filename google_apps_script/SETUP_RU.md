# Настройка Google Apps Script для Calendar Bot

## Шаг 1: Создать скрипт

1. Открой https://script.google.com
2. Нажми **"Новый проект"** (New project)
3. Удали всё содержимое файла `Code.gs`
4. Скопируй и вставь содержимое файла `Code.gs` из этой папки
5. **ВАЖНО:** В строке `var SECRET_TOKEN = 'CHANGE_ME_TO_RANDOM_STRING';` замени `CHANGE_ME_TO_RANDOM_STRING` на случайную строку (например, сгенерируй на https://passwordsgenerator.net)
6. Запомни этот токен — его нужно будет указать в `.env` файле бота как `GOOGLE_SCRIPT_TOKEN`

## Шаг 2: Развернуть как веб-приложение

1. Нажми **"Развернуть"** → **"Новое развёртывание"** (Deploy → New deployment)
2. Нажми на шестерёнку и выбери **"Веб-приложение"** (Web app)
3. Настройки:
   - **Описание**: Calendar Bot API
   - **Запуск от имени**: **Я** (Me)
   - **Доступ**: **Все** (Anyone)
4. Нажми **"Развернуть"** (Deploy)
5. **Разреши доступ** к Google Календарю когда появится запрос
6. Скопируй **URL веб-приложения** — это будет `GOOGLE_SCRIPT_URL` в `.env`

## Шаг 3: Проверить работу

Открой в браузере (заменив значения):
```
ВАШ_URL?action=getTodayEvents&token=ВАШ_ТОКЕН
```

Должен вернуться JSON с событиями на сегодня.

## Шаг 4: Обновление скрипта

Если нужно обновить код:
1. Внеси изменения в код
2. **Развернуть** → **Управление развёртываниями** → **Редактировать** → выбери **"Новая версия"** → **Развернуть**

URL остаётся тем же.
